AWS Item Tracker

The AWS Item Tracker application is a sample application that uses multiple AWS services and the Java V2 API. Collecting and working with items has never been easier! Simply perform these steps:

  1. Enter work items into the system by choosing the Add Items menu item. Fill in the form and then choose Create Item.
  2. The AWS Item Tracker application stores the data by using the Amazon Relational Database Service (Amazon RDS).
  3. You can view all of your items by choosing the Get Items menu item. Next, choose Get Active Items in the dialog box.
  4. You can modify an Active Item by selecting an item in the table and then choosing Get Single Item. The item appears in the Modify Item section where you can modify the description or status.
  5. Modify the item and then choose Update Item. You cannot modify the ID value.
  6. You can archive any item by selecting the item and choosing Archive Item. Notice that the table is updated with only active items.
  7. You can display all archived items by choosing Get Archived Items. You cannot modify an archived item.
  8. You can send an email recipient an email message with a report attachment by selecting the email recipient from the dialog box and then choosing Send Report.Only Active data is sent in a report.
  9. The Amazon Simple Email Service is used to send an email with an Excel document to the selected email recipient.
